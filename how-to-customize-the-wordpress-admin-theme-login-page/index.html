How to Customize The WordPress Admin Theme & Login Page

Written by Omar Badran May 3, 2020

Lets face it, the default WordPress admin theme is a bit clunky and not very intuitive. And the WordPress admin theme is a key branding element that most site owners dont pay attention to.

Having a consistent look plays a crucial role in the user experience. And if you are a freelancer you can differentiate your self and impress your clients with a modern & easy to use dashboard.

In this guide, I will show you how you can easily customize and unclutter the WordPress interface for yourself or your clients by using a free (and premium) plugin.

Step 1: Install Cora

Cora is the best white-label WordPress admin theme available in the market. It has a free & premium version.
The free version comes with tones of features like fully customizing the admin menu, adding custom dashboard widgets. fully customizing the toolbar, .etc.
However, you can always upgrade to unlock more features like customing the login page, dark theme, and much more.

To install the plugin. Go to Plugins -> Add new -> Search for Cora and click install. then activate the plugin.

After you activate the plugin. It should take you to the settings page.

Step 2: Customize The Admin Menu

In the menu section, you will see all options related to the admin menu. here you can show or hide the logo, upload your custom logo. or just use the site name.

You can also edit each menu item. you can reorder (drag & drop), change the title, change the icon, or hide the item for specific user roles.

Step 3: Customize The Toolbar

In the toolbar section you can remove the default WordPress items, like the WordPress logo, Pending comments & updates, etc.

And as you can see you can also add new items, just click add item and configure it as you wish. you can add as many items as you want.

Step 4: Change The Theme

Here you can customize every part of the design. you can also choose a predefined theme. Cora currently comes with two, light and dark. We are adding a new theme after each update.

You can still make your own, just change the colors until it fits your needs. colors here change instantly so that you can see the change live. without having to reload the page. after you are done click save to keep the changes.

Step 5: Customize The WordPress Login Page

The login page is the first thing that the user sees before accessing the admin dashboard, so having a clean modern login page is a must for every website. In this section you can customize the login page to suite your needs. you can change the logo, the page layout. you can add a title and description if you want.

Step 6: Add Custom Dashboard Widgets

With Cora you can add unlimited custom dashboard widgets with few clicks. Go to -> general -> Dashboard widgets, And click add. put in the content you want and change the title. Click save, And you should see the newly created widgets in the dashboard.

If you are interested, We have already written an article on how to add custom dashboard widgets using pages created with Gutenberg.

And we are done!

Summary

In this tutorial you have learned how to customize the WordPress admin theme & login page. If you liked this post make sure to check our blog for more tutorials like this. And if you like the plugin you can help us by rating it in the WordPress repository page.
